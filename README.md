# CANOpNode_Sys v4.0 - 工业级重力卸载控制系统

## 系统概述

CANOpNode_Sys 是一个基于 **NiMotion SDK** 的工业级多设备控制系统，采用分层架构设计：

- **Nimotion CANopen伺服电机**（通过NiMotion SDK控制，100Hz PDO实时通信）
- **电源板**（UART/TTL串口控制电流输出 0-4A）
- **编码器**（RS485接口，Modbus RTU协议，地址2）
- **压力计**（RS485接口，Modbus RTU协议，地址1，3位小数精度）

## 版本历史

- **v4.0** (2026-05-12): NiMotion SDK集成，工业级CANopen主站控制
- **v3.0** (2026-05-11): 自定义SocketCAN实现
- **v2.0**: 基础多线程架构
- **v1.0**: 原型系统

### 2. 线程架构

| 线程名称 | 频率 | 优先级 | 功能描述 |
|---------|------|--------|---------|
| data_collection_thread | 100Hz | 高 | 采集所有传感器数据到共享缓冲区 |
| data_output_thread | 20Hz | 中 | 输出数据到共享内存供上位机读取 |
| command_handler_thread | 100Hz | 高 | 处理上位机命令，实时电机控制 |
| sensor_manager_thread | 50-100Hz | 中 | RS485传感器轮询读取 |
| sync_thread (SDK内部) | 100Hz | 实时 | CAN Sync信号发送 |
| pdo_thread (SDK内部) | 100Hz | 实时 | PDO数据收发 |

### 3. 数据流图

```
压力计 ───────────────┘           │
电源板 ──────────────────────────┘

```

## 核心特性

### 1. 工业级CANopen控制

- **NiMotion SDK集成**: 使用官方SDK替代自定义SocketCAN
- **100Hz PDO通信**: 实时速度/位置控制
- **100Hz Sync信号**: 确保同步模式稳定运行
- **自动状态机管理**: CiA 402标准状态转换
- **故障诊断**: 自动读取故障码和报警信息

### 2. 模块化API设计

```c
// 电机驱动API
motor_init(&motor, node_id, can_if);    // 初始化
motor_enable(&motor);                    // 使能 (CSV模式)
motor_set_velocity(&motor, 1000);        // 设置速度 (rpm)
motor_get_velocity(&motor, &actual_vel); // 读取实际速度
motor_disable(&motor);                   // 失能
motor_deinit(&motor);                    // 反初始化
```

### 3. 传感器管理

- **统一接口**: 所有传感器通过统一API访问
- **自动校准**: 编码器零点和压力传感器去皮
- **数据融合**: 100Hz传感器数据融合到共享缓冲区
- **错误处理**: 自动重试和错误统计

### 4. 重力卸载算法

- **PID控制**: 基于压力反馈的主动重力补偿
- **安全监控**: 实时安全边界检查
- **算法状态**: 完整的状态机管理
- **独立运行**: 算法和手动模式可独立运行

## 快速开始

### 1. 编译

```bash
cd CANOpNode_Sys
make clean
make
```

### 2. 运行

```bash
# 需要root权限访问CAN设备和串口
sudo ./bin/CANOpNode_Sys

# 禁用电机控制（仅传感器）
sudo ./bin/CANOpNode_Sys --no-motor
```

### 3. 系统检查

程序启动时会自动执行：
1. 硬件初始化（传感器、电源、电机）
2. 传感器校准（编码器零点、压力去皮）
3. 系统健康检查
4. 通信线程启动

## 硬件配置

### CAN接口

- **设备**: NiMotion SCM-USBCAN-A (USB转CAN)
- **波特率**: 1Mbps
- **节点ID**: 1
- **Sync周期**: 10ms (100Hz)
- **PDO周期**: 10ms (100Hz)

### RS485传感器

- **设备**: /dev/ttyUSB1
- **波特率**: 115200
- **编码器地址**: 2
- **压力计地址**: 1

### 电源板

- **设备**: /dev/ttyUSB0
- **波特率**: 115200
- **电流范围**: 0-4A
- **电压范围**: 0-48V
